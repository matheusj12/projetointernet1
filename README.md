# Almoxarifado Digital

Sistema de controle de estoque para projetos de infraestrutura.

## Funcionalidades
- 📊 Dashboard com visão geral do estoque
- 📦 Catálogo de compras com controle de orçamento
- 📥 Entradas de material (recebimento fracionado)
- 📤 Saídas/Cautelas com regra das duas pontas
- 🔄 Logística reversa (devoluções ao fornecedor)
- 👥 Gestão de equipe (técnicos e autorizadores)
- 📸 Registro fotográfico nos processos
- 📊 Exportação para Excel
- 📝 Log de auditoria completo
- 📱 Interface responsiva (desktop e mobile)

## Tecnologias
- React 19 + TypeScript + Vite
- Supabase (PostgreSQL + Storage)
- Vanilla CSS (dark mode)

## Deploy
- Vercel (automático via GitHub)
